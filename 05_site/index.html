03 · THE SYSTEM

Three layers.
Decoupled by design.

We split the protocol into physical, application, and chain layers so any failure is contained. One tap to unlock a locker triggers a seven-step journey that clears end-to-end in 1.2 seconds.

L3 · PHYSICAL

IoT & lockers

NFC reader · NB-IoT gateway · MQTT broker. Sub-100ms unlock latency, sub-1W power draw.

L2 · APPLICATION

Wallet & backend

Consumer wallet, Y Carry operations dashboard, merchant POS adapters. React + TypeScript.

L1 · CHAIN

Solana settlement

SPL token · Metaplex metadata · five custom contracts. 65,000 TPS headroom, sub-cent fees.
