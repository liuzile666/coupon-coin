# 02 · White Paper

The public-facing version of the dissertation, plus the speaker script for investor presentations.

## Files

| File | What it is |
|---|---|
| `Coupon_Coin_White_Paper_v1.0.docx` | 21 KB public white paper — distilled thesis, no academic apparatus |
| `Speech_Script.md` | 28 KB speaker script used at industry talks (Apr 2026) |

## Difference from the dissertation

| | Dissertation | White paper |
|---|---|---|
| Audience | Academic examiners | Investors, partners, community |
| Length | ~14,000 words | ~4,000 words |
| Citations | Full academic referencing | Removed |
| Tone | Neutral, hedged | Confident, forward-leaning |
| Focus | Process + evidence | Value proposition + roadmap |

## When to use which

- **Share the white paper** with a potential partner, a crypto-aware friend, or anyone who asks "what does your project do?"
- **Share the dissertation** with an examiner, a researcher, or someone who wants to pressure-test every claim.
- **Use the speech script** as ready-made talking points for a 15-minute talk — it is structured as three acts (problem / solution / ask).
